# ICM20948 DRIVER

## 1. Giới thiệu

File này mô tả driver ICM20948 được viết cho vi điều khiển STM32F411CEU6.

Driver được xây dựng theo phương pháp Bare-metal, sử dụng thư viện SPI để giao tiếp với cảm biến ICM20948.

Các file liên quan:

- `icm20948.c`: Chứa phần hiện thực các hàm ICM20948.
- `icm20948.h`: Chứa khai báo hàm, kiểu dữ liệu và các định nghĩa cần thiết.
- `spi.h` : Giao tiếp SPI với cảm biến ICM20948.
- `delay.h` : Tạo delay trong quá trình khởi tạo và Calibration cảm biến.

## 2. Phần cứng sử dụng

Vi điều khiển: `STM32F411CEU6`

Cảm biến: `ICM20948`

Giao thức: `SPI`

Dữ liệu đọc: `Accelerometer`  `Gyroscope`  `Temperature`

## 3. Cấu hình ICM20948

Driver cấu hình ICM20948 hoạt động qua giao tiếp SPI.

Các thông số:

- WHO_AM_I: `0xEA`
- Accelerometer Full Scale: `±16G`
- Gyroscope Full Scale: `±2000DPS`
- Accelerometer DLPF: `246Hz`
- Gyroscope DLPF: `196Hz`
- Accelerometer ODR Divider: `1`
- Gyroscope ODR Divider: `1`
- I2C Interface: `Disable`

## 4. Các thanh ghi sử dụng

### BANK 0

- ICM20948_REG_WHO_AM_I
- ICM20948_REG_USER_CTRL
- ICM20948_REG_PWR_MGMT_1
- ICM20948_REG_PWR_MGMT_2
- ICM20948_REG_ACCEL_XOUT_H
- ICM20948_REG_REG_BANK_SEL

### BANK 2

- ICM20948_REG_GYRO_SMPLRT_DIV
- ICM20948_REG_GYRO_CONFIG_1
- ICM20948_REG_ACCEL_SMPLRT_DIV_1
- ICM20948_REG_ACCEL_SMPLRT_DIV_2
- ICM20948_REG_ACCEL_CONFIG

## 5. ICM20948.h

- Tạo kiểu dữ liệu trả về trạng thái `ICM20948_StatusTypeDef`

- Tạo kiểu dữ liệu cấu hình Full Scale Accelerometer `ICM20948_AccelFullScale_t`

- Tạo kiểu dữ liệu cấu hình Full Scale Gyroscope `ICM20948_GyroFullScale_t`

- Tạo kiểu dữ liệu cấu hình ODR `ICM20948_ODR_t`

- Tạo kiểu dữ liệu lưu dữ liệu thô `ICM20948_RawData_t`

- Tạo kiểu dữ liệu lưu dữ liệu cảm biến `ICM20948_Data_t`

- Tạo kiểu dữ liệu lưu thông số Calibration Gyroscope `ICM20948_GyroCalibration_t`

- Khai báo các hàm sẽ được gọi trong thư viện khác :

    + `ICM20948_StatusTypeDef ICM20948_Init(void)` : hàm khởi tạo và cấu hình ICM20948

    + `ICM20948_StatusTypeDef ICM20948_ReadWhoAmI(uint8_t *who_am_i)` : Hàm đọc giá trị WHO_AM_I

    + `ICM20948_StatusTypeDef ICM20948_SetAccelConfig(ICM20948_AccelFullScale_t full_scale, ICM20948_ODR_t odr)` : Hàm cấu hình Accelerometer

    + `ICM20948_StatusTypeDef ICM20948_SetGyroConfig(ICM20948_GyroFullScale_t full_scale, ICM20948_ODR_t odr)` : Hàm cấu hình Gyroscope

    + `ICM20948_StatusTypeDef ICM20948_CalibrateGyro(uint16_t sample_count)` : Hàm Calibration Gyroscope

    + `void ICM20948_GetGyroCalibration(ICM20948_GyroCalibration_t *calibration)` : Hàm lấy thông số Calibration Gyroscope

    + `ICM20948_StatusTypeDef ICM20948_ReadRawData(ICM20948_RawData_t *raw_data)` : Hàm đọc dữ liệu thô Accelerometer, Gyroscope và Temperature

    + `ICM20948_StatusTypeDef ICM20948_ConvertRawToSensorData(const ICM20948_RawData_t *raw_data, ICM20948_Data_t *sensor_data)` : Hàm chuyển dữ liệu thô sang dữ liệu cảm biến

    + `ICM20948_StatusTypeDef ICM20948_ReadSensorData(ICM20948_Data_t *sensor_data)` : Hàm đọc dữ liệu cảm biến

## 6. ICM20948.c

- Thực hiện các hàm ICM20948: 

* `ICM20948_ConvertSPIStatus(SPI_Status_t status)` : Hàm chuyển trạng thái trả về của SPI sang trạng thái trả về của ICM20948

    + `status` : trạng thái trả về từ thư viện SPI

* `ICM20948_WriteReg(uint8_t reg, uint8_t data)` : Hàm ghi một byte dữ liệu vào một thanh ghi cụ thể của ICM20948

    + `reg` : địa chỉ thanh ghi cần ghi dữ liệu

    + `data` : byte dữ liệu cần ghi

    + `Ví dụ` : `ICM20948_WriteReg(ICM20948_REG_PWR_MGMT_1, ICM20948_DEVICE_RESET)`

* `ICM20948_ReadReg(uint8_t reg, uint8_t *data)` : Hàm đọc một byte dữ liệu từ một thanh ghi cụ thể của ICM20948

    + `reg` : địa chỉ thanh ghi cần đọc dữ liệu

    + `data` : địa chỉ biến dùng để lưu dữ liệu đọc được

    + `Ví dụ` : `ICM20948_ReadReg(ICM20948_REG_WHO_AM_I, who_am_i)`

* `ICM20948_ReadRegs(uint8_t reg, uint8_t *data, uint8_t len)` : Hàm đọc một hoặc nhiều byte dữ liệu từ một thanh ghi cụ thể của ICM20948

    + `reg` : địa chỉ thanh ghi bắt đầu đọc dữ liệu

    + `data` : địa chỉ vùng nhớ dùng để lưu dữ liệu đọc được

    + `len` : số byte dữ liệu cần đọc

    + `Ví dụ` : `ICM20948_ReadRegs(ICM20948_REG_ACCEL_XOUT_H, raw, 14)` đọc 14 byte dữ liệu bắt đầu từ thanh ghi `ICM20948_REG_ACCEL_XOUT_H`

* `ICM20948_SelectBank(uint8_t bank)` : Hàm chọn Register Bank của ICM20948

    + `bank` : Bank cần chọn

    + `Ví dụ` : `ICM20948_SelectBank(ICM20948_BANK0)`

* `ICM20948_CheckID(void)` : Hàm kiểm tra giá trị WHO_AM_I của ICM20948

* `ICM20948_Reset(void)` : Hàm reset cảm biến ICM20948

* `ICM20948_GetAccelSensitivity(void)` : Hàm lấy độ nhạy Accelerometer tương ứng với Full Scale đang cấu hình

* `ICM20948_GetGyroSensitivity(void)` : Hàm lấy độ nhạy Gyroscope tương ứng với Full Scale đang cấu hình

* `ICM20948_Init(void)` : Hàm khởi tạo và cấu hình ICM20948

    + Khởi tạo giao tiếp SPI

    + Chọn Register Bank 0

    + Kiểm tra WHO_AM_I

    + Reset ICM20948

    + Tắt giao tiếp I2C

    + Cấu hình nguồn clock

    + Bật Accelerometer và Gyroscope

    + Cấu hình Gyroscope Full Scale ±2000DPS và ODR Divider 1

    + Cấu hình Accelerometer Full Scale ±16G và ODR Divider 1

    + Không nhận tham số đầu vào

    + `Ví dụ` : `ICM20948_Init()`

* `ICM20948_ReadWhoAmI(uint8_t *who_am_i)` : Hàm đọc giá trị WHO_AM_I của ICM20948

    + `who_am_i` : địa chỉ biến dùng để lưu giá trị WHO_AM_I

    + `Ví dụ` : `ICM20948_ReadWhoAmI(&who_am_i)`

* `ICM20948_SetAccelConfig(ICM20948_AccelFullScale_t full_scale, ICM20948_ODR_t odr)` : Hàm cấu hình Full Scale, ODR và DLPF cho Accelerometer

    + `full_scale` : dải đo Full Scale của Accelerometer

    + `odr` : giá trị chia Output Data Rate của Accelerometer

    + `Ví dụ` : `ICM20948_SetAccelConfig(ICM20948_ACCEL_FS_16G, ICM20948_ODR_DIV_1)`

* `ICM20948_SetGyroConfig(ICM20948_GyroFullScale_t full_scale, ICM20948_ODR_t odr)` : Hàm cấu hình Full Scale, ODR và DLPF cho Gyroscope

    + `full_scale` : dải đo Full Scale của Gyroscope

    + `odr` : giá trị chia Output Data Rate của Gyroscope

    + `Ví dụ` : `ICM20948_SetGyroConfig(ICM20948_GYRO_FS_2000DPS, ICM20948_ODR_DIV_1)`

* `ICM20948_ReadRawData(ICM20948_RawData_t *raw_data)` : Hàm đọc dữ liệu thô Accelerometer, Gyroscope và Temperature

    + `raw_data` : địa chỉ vùng nhớ dùng để lưu dữ liệu thô Accelerometer, Gyroscope và Temperature

    + `Ví dụ` : `ICM20948_ReadRawData(&raw_data)`

* `ICM20948_CalibrateGyro(uint16_t sample_count)` : Hàm Calibration Gyroscope bằng cách lấy trung bình nhiều mẫu dữ liệu thô để tính Offset

    + `sample_count` : số mẫu dữ liệu dùng để Calibration

    + `Ví dụ` : `ICM20948_CalibrateGyro(200)`

* `ICM20948_GetGyroCalibration(ICM20948_GyroCalibration_t *calibration)` : Hàm lấy thông số Offset Calibration của Gyroscope

    + `calibration` : địa chỉ vùng nhớ dùng để lưu thông số Calibration Gyroscope

    + `Ví dụ` : `ICM20948_GetGyroCalibration(&calibration)`

* `ICM20948_ConvertRawToSensorData(const ICM20948_RawData_t *raw_data, ICM20948_Data_t *sensor_data)` : Hàm chuyển dữ liệu thô thành giá trị gia tốc, vận tốc góc và nhiệt độ

    + `raw_data` : địa chỉ vùng dữ liệu thô Accelerometer, Gyroscope và Temperature

    + `sensor_data` : địa chỉ vùng nhớ dùng để lưu dữ liệu sau khi chuyển đổi

    + `Ví dụ` : `ICM20948_ConvertRawToSensorData(&raw_data, &sensor_data)`

* `ICM20948_ReadSensorData(ICM20948_Data_t *sensor_data)` : Hàm đọc dữ liệu thô và chuyển sang giá trị gia tốc, vận tốc góc và nhiệt độ

    + `sensor_data` : địa chỉ vùng nhớ dùng để lưu dữ liệu cảm biến

    + `Ví dụ` : `ICM20948_ReadSensorData(&sensor_data)`
