# BMP280 DRIVER

## 1. Giới thiệu

File này mô tả driver BMP280 được viết cho vi điều khiển STM32F411CEU6.

Driver được xây dựng theo phương pháp Bare-metal, sử dụng thư viện I2C để giao tiếp với cảm biến BMP280.

Các file liên quan:

- `bmp280.c`: Chứa phần hiện thực các hàm BMP280.
- `bmp280.h`: Chứa khai báo hàm, kiểu dữ liệu và các định nghĩa cần thiết.
- `i2c.h` : Giao tiếp I2C với cảm biến BMP280.
- `delay.h` : Tạo delay trong quá trình khởi tạo cảm biến.

## 2. Phần cứng sử dụng

Vi điều khiển: `STM32F411CEU6`

Cảm biến: `BMP280`

Giao thức: `I2C`

Địa chỉ I2C: `0x76` hoặc `0x77`

## 3. Cấu hình BMP280

Driver cấu hình BMP280 hoạt động ở chế độ Normal Mode.

Các thông số:

- Địa chỉ I2C: `0x76` hoặc `0x77`
- Chip ID: `0x58`
- Temperature Oversampling: `x1`
- Pressure Oversampling: `x4`
- Mode: `Normal Mode`
- Standby Time: `0.5ms`
- Filter: `4`
- SPI 3-wire: `Disable`

## 5. BMP280.h

- Tạo kiểu dữ liệu trả về trạng thái `BMP280_StatusTypeDef`

- Khai báo các hàm sẽ được gọi trong thư viện khác :

    + `BMP280_StatusTypeDef BMP280_Init(void)` : hàm khởi tạo và cấu hình BMP280

    + `BMP280_StatusTypeDef BMP280_SetReferencePressure(void)` : Hàm lấy áp suất tham chiếu ban đầu

    + `BMP280_StatusTypeDef BMP280_ReadTemperatureAndPressureAndAltitude(float *temperature, float *pressure, float *altitude)` : Hàm đọc nhiệt độ, áp suất và độ cao

## 6. BMP280.c

- Thực hiện các hàm BMP280: 

* `BMP280_WriteReg(uint8_t reg, uint8_t data)` : Hàm ghi một byte dữ liệu vào một thanh ghi cụ thể của BMP280

    + `reg` : địa chỉ thanh ghi cần ghi dữ liệu

    + `data` : byte dữ liệu cần ghi

    + `Ví dụ` : `BMP280_WriteReg(BMP280_REG_RESET, 0xB6)`

* `BMP280_ReadReg(uint8_t reg, uint8_t *data)` : Hàm đọc một byte dữ liệu từ một thanh ghi cụ thể của BMP280

    + `reg` : địa chỉ thanh ghi cần đọc dữ liệu

    + `data` : địa chỉ biến dùng để lưu dữ liệu đọc được

    + `Ví dụ` : `BMP280_ReadReg(BMP280_REG_ID, &BMP280_ID)`

* `BMP280_ReadRegs(uint8_t reg, uint8_t *data, uint8_t len)` : Hàm đọc một hoặc nhiều byte dữ liệu từ một thanh ghi cụ thể của BMP280

    + `reg` : địa chỉ thanh ghi bắt đầu đọc dữ liệu

    + `data` : địa chỉ vùng nhớ dùng để lưu dữ liệu đọc được

    + `len` : số byte dữ liệu cần đọc

    + `Ví dụ` : `BMP280_ReadRegs(BMP280_REG_PRESS_MSB, rawData, 6)` đọc 6 byte dữ liệu bắt đầu từ thanh ghi `BMP280_REG_PRESS_MSB`

* `BMP280_CheckID(void)` : Hàm kiểm tra địa chỉ I2C và Chip ID của cảm biến BMP280

* `BMP280_Reset(void)` : Hàm reset cảm biến BMP280 bằng cách ghi giá trị `0xB6` vào thanh ghi RESET

* `BMP280_ReadCalibration(void)` : Hàm đọc các hệ số Calibration dùng để xử lý dữ liệu nhiệt độ và áp suất

* `BMP280_SetCtrlMeas(void)` : Hàm cấu hình thanh ghi CTRL_MEAS với Temperature Oversampling x1, Pressure Oversampling x4 và Normal Mode

* `BMP280_SetConfig(void)` : Hàm cấu hình thanh ghi CONFIG với Standby Time 0.5ms, Filter 4 và tắt SPI 3-wire

* `BMP280_Init(void)` : Hàm khởi tạo và cấu hình BMP280

    + Khởi tạo giao tiếp I2C

    + Kiểm tra địa chỉ I2C và Chip ID của BMP280

    + Reset BMP280

    + Đọc các hệ số Calibration

    + Cấu hình thanh ghi CONFIG

    + Cấu hình thanh ghi CTRL_MEAS

    + Không nhận tham số đầu vào

    + `Ví dụ` : `BMP280_Init()`

* `BMP280_ReadRawData(void)` : Hàm đọc dữ liệu thô nhiệt độ và áp suất từ BMP280

* `BMP280_CompensateTemperature(void)` : Hàm sử dụng dữ liệu thô và hệ số Calibration để tính nhiệt độ

* `BMP280_CompensatePressure(void)` : Hàm sử dụng dữ liệu thô, hệ số Calibration và t_fine để tính áp suất

* `BMP280_SetReferencePressure(void)` : Hàm lấy áp suất hiện tại làm áp suất tham chiếu P0 để tính độ cao

    + Không nhận tham số đầu vào

    + `Ví dụ` : `BMP280_SetReferencePressure()`

* `BMP280_ReadTemperatureAndPressureAndAltitude(float *temperature, float *pressure, float *altitude)` : Hàm đọc và tính nhiệt độ, áp suất và độ cao từ BMP280

    + `temperature` : địa chỉ biến dùng để lưu giá trị nhiệt độ

    + `pressure` : địa chỉ biến dùng để lưu giá trị áp suất

    + `altitude` : địa chỉ biến dùng để lưu giá trị độ cao

    + `Ví dụ` : `BMP280_ReadTemperatureAndPressureAndAltitude(&temperature, &pressure, &altitude)`
