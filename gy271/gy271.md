# GY271 DRIVER

## 1. Giới thiệu

File này mô tả driver GY271 được viết cho vi điều khiển STM32F411CEU6.

Driver được xây dựng theo phương pháp Bare-metal, sử dụng thư viện I2C để giao tiếp với cảm biến GY271.

Các file liên quan:

- `gy271.c`: Chứa phần hiện thực các hàm GY271.
- `gy271.h`: Chứa khai báo hàm, kiểu dữ liệu và các định nghĩa cần thiết.
- `i2c.h` : Giao tiếp I2C với cảm biến GY271.
- `delay.h` : Tạo delay trong quá trình khởi tạo và chờ dữ liệu cảm biến.

## 2. Phần cứng sử dụng

Vi điều khiển: `STM32F411CEU6`

Cảm biến: `GY271`

Chip hỗ trợ: `QMC5883L`  `HMC5883L`

Giao thức: `I2C`

Địa chỉ I2C QMC5883L: `0x0D`

Địa chỉ I2C HMC5883L: `0x1E`

## 3. Cấu hình GY271

Driver tự kiểm tra và cấu hình GY271 theo loại chip QMC5883L hoặc HMC5883L.

Các thông số:

- QMC5883L:
    + Địa chỉ I2C: `0x0D`
    + OSR: `512`
    + Range: `8G`
    + ODR: `200Hz`
    + Mode: `Continuous`

- HMC5883L:
    + Địa chỉ I2C: `0x1E`
    + Samples Averaged: `8`
    + Data Output Rate: `15Hz`
    + Gain: `1.3G`
    + Mode: `Continuous`

## 4. Các thanh ghi sử dụng

### QMC5883L

- GY271_QMC_REG_X_LSB
- GY271_QMC_REG_STATUS
- GY271_QMC_REG_CTRL1
- GY271_QMC_REG_CTRL2
- GY271_QMC_REG_SET_RESET
- GY271_QMC_REG_CHIP_ID

### HMC5883L

- GY271_HMC_REG_CONFIG_A
- GY271_HMC_REG_CONFIG_B
- GY271_HMC_REG_MODE
- GY271_HMC_REG_X_MSB
- GY271_HMC_REG_STATUS
- GY271_HMC_REG_ID_A

## 5. GY271.h

- Tạo kiểu dữ liệu trả về trạng thái `GY271_StatusTypeDef`

- Tạo kiểu dữ liệu lưu dữ liệu thô `GY271_RawData`

- Tạo kiểu dữ liệu lưu dữ liệu từ trường `GY271_MagneticData`

- Khai báo các hàm sẽ được gọi trong thư viện khác :

    + `GY271_StatusTypeDef GY271_Init(void)` : hàm khởi tạo và cấu hình GY271

    + `GY271_StatusTypeDef GY271_ReadRawData(GY271_RawData *rawData)` : Hàm đọc dữ liệu thô 3 trục X, Y, Z

    + `GY271_StatusTypeDef GY271_ReadMagneticField(GY271_MagneticData *magData)` : Hàm đọc giá trị từ trường 3 trục X, Y, Z

    + `GY271_StatusTypeDef GY271_ReadHeading(float *heading)` : Hàm đọc góc Heading

    + `GY271_StatusTypeDef GY271_ReadData(GY271_RawData *rawData, GY271_MagneticData *magData, float *heading)` : Hàm đọc dữ liệu thô, từ trường và góc Heading

## 6. GY271.c

- Thực hiện các hàm GY271: 

* `GY271_WriteReg(uint8_t reg, uint8_t data)` : Hàm ghi một byte dữ liệu vào một thanh ghi cụ thể của GY271

    + `reg` : địa chỉ thanh ghi cần ghi dữ liệu

    + `data` : byte dữ liệu cần ghi

    + `Ví dụ` : `GY271_WriteReg(GY271_QMC_REG_CTRL2, 0x80)`

* `GY271_ReadReg(uint8_t reg, uint8_t *data)` : Hàm đọc một byte dữ liệu từ một thanh ghi cụ thể của GY271

    + `reg` : địa chỉ thanh ghi cần đọc dữ liệu

    + `data` : địa chỉ biến dùng để lưu dữ liệu đọc được

    + `Ví dụ` : `GY271_ReadReg(GY271_QMC_REG_STATUS, &status)`

* `GY271_ReadRegs(uint8_t reg, uint8_t *data, uint8_t len)` : Hàm đọc một hoặc nhiều byte dữ liệu từ một thanh ghi cụ thể của GY271

    + `reg` : địa chỉ thanh ghi bắt đầu đọc dữ liệu

    + `data` : địa chỉ vùng nhớ dùng để lưu dữ liệu đọc được

    + `len` : số byte dữ liệu cần đọc

    + `Ví dụ` : `GY271_ReadRegs(GY271_QMC_REG_X_LSB, raw, 6)` đọc 6 byte dữ liệu bắt đầu từ thanh ghi `GY271_QMC_REG_X_LSB`

* `GY271_CheckID(void)` : Hàm kiểm tra địa chỉ I2C và xác định chip QMC5883L hoặc HMC5883L

* `GY271_Reset(void)` : Hàm reset cảm biến QMC5883L bằng cách ghi giá trị `0x80` vào thanh ghi CTRL2

* `GY271_SetConfig(void)` : Hàm cấu hình GY271 theo loại chip QMC5883L hoặc HMC5883L

* `GY271_WaitDataReady(void)` : Hàm chờ dữ liệu cảm biến sẵn sàng để đọc

* `GY271_ConvertRawToMagnetic(const GY271_RawData *rawData, GY271_MagneticData *magData)` : Hàm chuyển dữ liệu thô sang giá trị từ trường

    + `rawData` : địa chỉ vùng dữ liệu thô X, Y, Z

    + `magData` : địa chỉ vùng nhớ dùng để lưu giá trị từ trường X, Y, Z

* `GY271_CalculateHeading(const GY271_MagneticData *magData, float *heading)` : Hàm tính góc Heading từ dữ liệu từ trường trục X và Y

    + `magData` : địa chỉ vùng dữ liệu từ trường X, Y, Z

    + `heading` : địa chỉ biến dùng để lưu góc Heading

* `GY271_Init(void)` : Hàm khởi tạo và cấu hình GY271

    + Khởi tạo giao tiếp I2C

    + Kiểm tra địa chỉ I2C và xác định chip QMC5883L hoặc HMC5883L

    + Reset cảm biến

    + Cấu hình cảm biến theo loại chip

    + Không nhận tham số đầu vào

    + `Ví dụ` : `GY271_Init()`

* `GY271_ReadRawData(GY271_RawData *rawData)` : Hàm đọc dữ liệu thô 3 trục X, Y, Z từ GY271

    + `rawData` : địa chỉ vùng nhớ dùng để lưu dữ liệu thô X, Y, Z

    + `Ví dụ` : `GY271_ReadRawData(&rawData)`

* `GY271_ReadMagneticField(GY271_MagneticData *magData)` : Hàm đọc và chuyển dữ liệu thô sang giá trị từ trường 3 trục X, Y, Z

    + `magData` : địa chỉ vùng nhớ dùng để lưu giá trị từ trường X, Y, Z

    + `Ví dụ` : `GY271_ReadMagneticField(&magData)`

* `GY271_ReadHeading(float *heading)` : Hàm đọc và tính góc Heading từ GY271

    + `heading` : địa chỉ biến dùng để lưu góc Heading

    + `Ví dụ` : `GY271_ReadHeading(&heading)`

* `GY271_ReadData(GY271_RawData *rawData, GY271_MagneticData *magData, float *heading)` : Hàm đọc dữ liệu thô, chuyển sang giá trị từ trường và tính góc Heading

    + `rawData` : địa chỉ vùng nhớ dùng để lưu dữ liệu thô X, Y, Z

    + `magData` : địa chỉ vùng nhớ dùng để lưu giá trị từ trường X, Y, Z

    + `heading` : địa chỉ biến dùng để lưu góc Heading

    + `Ví dụ` : `GY271_ReadData(&rawData, &magData, &heading)`
