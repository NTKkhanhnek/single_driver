# ICM42688 DRIVER

## 1. Giới thiệu

File này mô tả driver ICM42688 được viết cho vi điều khiển STM32F411CEU6.

Driver được xây dựng theo phương pháp Bare-metal, sử dụng thư viện SPI để giao tiếp với cảm biến ICM42688.

Các file liên quan:

- `icm42688.c`: Chứa phần hiện thực các hàm ICM42688.
- `icm42688.h`: Chứa khai báo hàm, kiểu dữ liệu và các định nghĩa cần thiết.
- `spi.h` : Giao tiếp SPI với cảm biến ICM42688.
- `delay.h` : Tạo delay trong quá trình khởi tạo cảm biến.

## 2. Phần cứng sử dụng

Vi điều khiển: `STM32F411CEU6`

Cảm biến: `ICM42688`

Giao thức: `SPI`

Dữ liệu đọc: `Accelerometer`  `Gyroscope`  `Temperature`

## 3. Cấu hình ICM42688

Driver cấu hình ICM42688 hoạt động qua giao tiếp SPI.

Các thông số:

- WHO_AM_I: `0x47`
- Accelerometer Full Scale: `±16G`
- Gyroscope Full Scale: `±2000DPS`
- Accelerometer ODR: `1kHz`
- Gyroscope ODR: `1kHz`
- Filter Bandwidth: `ODR/4`
- Accelerometer Mode: `Low Noise`
- Gyroscope Mode: `Low Noise`

## 4. Các thanh ghi sử dụng

### BANK 0

- ICM42688_REG_DEVICE_CONFIG
- ICM42688_REG_TEMP_DATA1

- ICM42688_REG_SIGNAL_PATH_RESET
- ICM42688_REG_PWR_MGMT0
- ICM42688_REG_GYRO_CONFIG0
- ICM42688_REG_ACCEL_CONFIG0
- ICM42688_REG_GYRO_ACCEL_CONFIG0
- ICM42688_REG_WHO_AM_I
- ICM42688_REG_BANK_SEL

## 5. ICM42688.h

- Tạo kiểu dữ liệu trả về trạng thái `ICM42688_StatusTypeDef`

- Tạo kiểu dữ liệu cấu hình Full Scale Accelerometer `ICM42688_AccelFullScale_t`

- Tạo kiểu dữ liệu cấu hình Full Scale Gyroscope `ICM42688_GyroFullScale_t`

- Tạo kiểu dữ liệu cấu hình ODR `ICM42688_ODR_t`

- Tạo kiểu dữ liệu lưu dữ liệu thô `ICM42688_RawData_t`

- Tạo kiểu dữ liệu lưu dữ liệu cảm biến `ICM42688_Data_t`

- Khai báo các hàm sẽ được gọi trong thư viện khác :

    + `ICM42688_StatusTypeDef ICM42688_Init(void)` : hàm khởi tạo và cấu hình ICM42688

    + `ICM42688_StatusTypeDef ICM42688_ReadWhoAmI(uint8_t *who_am_i)` : Hàm đọc giá trị WHO_AM_I

    + `ICM42688_StatusTypeDef ICM42688_SetAccelConfig(ICM42688_AccelFullScale_t full_scale, ICM42688_ODR_t odr)` : Hàm cấu hình Accelerometer

    + `ICM42688_StatusTypeDef ICM42688_SetGyroConfig(ICM42688_GyroFullScale_t full_scale, ICM42688_ODR_t odr)` : Hàm cấu hình Gyroscope

    + `ICM42688_StatusTypeDef ICM42688_ReadRawData(ICM42688_RawData_t *raw_data)` : Hàm đọc dữ liệu thô Accelerometer, Gyroscope và Temperature

    + `ICM42688_StatusTypeDef ICM42688_ConvertRawToSensorData(const ICM42688_RawData_t *raw_data, ICM42688_Data_t *sensor_data)` : Hàm chuyển dữ liệu thô sang dữ liệu cảm biến

    + `ICM42688_StatusTypeDef ICM42688_ReadSensorData(ICM42688_Data_t *sensor_data)` : Hàm đọc dữ liệu cảm biến

## 6. ICM42688.c

- Thực hiện các hàm ICM42688: 

* `ICM42688_ConvertSPIStatus(SPI_Status_t status)` : Hàm chuyển trạng thái trả về của SPI sang trạng thái trả về của ICM42688

    + `status` : trạng thái trả về từ thư viện SPI

* `ICM42688_WriteReg(uint8_t reg, uint8_t data)` : Hàm ghi một byte dữ liệu vào một thanh ghi cụ thể của ICM42688

    + `reg` : địa chỉ thanh ghi cần ghi dữ liệu

    + `data` : byte dữ liệu cần ghi

    + `Ví dụ` : `ICM42688_WriteReg(ICM42688_REG_DEVICE_CONFIG, ICM42688_SOFT_RESET)`

* `ICM42688_ReadReg(uint8_t reg, uint8_t *data)` : Hàm đọc một byte dữ liệu từ một thanh ghi cụ thể của ICM42688

    + `reg` : địa chỉ thanh ghi cần đọc dữ liệu

    + `data` : địa chỉ biến dùng để lưu dữ liệu đọc được

    + `Ví dụ` : `ICM42688_ReadReg(ICM42688_REG_WHO_AM_I, who_am_i)`

* `ICM42688_ReadRegs(uint8_t reg, uint8_t *data, uint8_t len)` : Hàm đọc một hoặc nhiều byte dữ liệu từ một thanh ghi cụ thể của ICM42688

    + `reg` : địa chỉ thanh ghi bắt đầu đọc dữ liệu

    + `data` : địa chỉ vùng nhớ dùng để lưu dữ liệu đọc được

    + `len` : số byte dữ liệu cần đọc

    + `Ví dụ` : `ICM42688_ReadRegs(ICM42688_REG_TEMP_DATA1, raw, 14)` đọc 14 byte dữ liệu bắt đầu từ thanh ghi `ICM42688_REG_TEMP_DATA1`

* `ICM42688_SelectBank(uint8_t bank)` : Hàm chọn Register Bank của ICM42688

    + `bank` : Bank cần chọn

    + `Ví dụ` : `ICM42688_SelectBank(ICM42688_BANK0)`

* `ICM42688_CheckID(void)` : Hàm kiểm tra giá trị WHO_AM_I của ICM42688

* `ICM42688_Reset(void)` : Hàm reset cảm biến ICM42688

* `ICM42688_SetFilterBandwidth(void)` : Hàm cấu hình Filter Bandwidth bằng ODR/4 cho Accelerometer và Gyroscope

* `ICM42688_GetAccelSensitivity(void)` : Hàm lấy độ nhạy Accelerometer tương ứng với Full Scale đang cấu hình

* `ICM42688_GetGyroSensitivity(void)` : Hàm lấy độ nhạy Gyroscope tương ứng với Full Scale đang cấu hình

* `ICM42688_Init(void)` : Hàm khởi tạo và cấu hình ICM42688

    + Khởi tạo giao tiếp SPI

    + Chọn Register Bank 0

    + Kiểm tra WHO_AM_I

    + Reset ICM42688

    + Cấu hình Gyroscope Full Scale ±2000DPS và ODR 1kHz

    + Cấu hình Accelerometer Full Scale ±16G và ODR 1kHz

    + Cấu hình Filter Bandwidth bằng ODR/4

    + Bật Accelerometer và Gyroscope ở chế độ Low Noise

    + Không nhận tham số đầu vào

    + `Ví dụ` : `ICM42688_Init()`

* `ICM42688_ReadWhoAmI(uint8_t *who_am_i)` : Hàm đọc giá trị WHO_AM_I của ICM42688

    + `who_am_i` : địa chỉ biến dùng để lưu giá trị WHO_AM_I

    + `Ví dụ` : `ICM42688_ReadWhoAmI(&who_am_i)`

* `ICM42688_SetAccelConfig(ICM42688_AccelFullScale_t full_scale, ICM42688_ODR_t odr)` : Hàm cấu hình Full Scale và ODR cho Accelerometer

    + `full_scale` : dải đo Full Scale của Accelerometer

    + `odr` : Output Data Rate của Accelerometer

    + `Ví dụ` : `ICM42688_SetAccelConfig(ICM42688_ACCEL_FS_16G, ICM42688_ODR_1KHZ)`

* `ICM42688_SetGyroConfig(ICM42688_GyroFullScale_t full_scale, ICM42688_ODR_t odr)` : Hàm cấu hình Full Scale và ODR cho Gyroscope

    + `full_scale` : dải đo Full Scale của Gyroscope

    + `odr` : Output Data Rate của Gyroscope

    + `Ví dụ` : `ICM42688_SetGyroConfig(ICM42688_GYRO_FS_2000DPS, ICM42688_ODR_1KHZ)`

* `ICM42688_ReadRawData(ICM42688_RawData_t *raw_data)` : Hàm đọc dữ liệu thô Accelerometer, Gyroscope và Temperature

    + `raw_data` : địa chỉ vùng nhớ dùng để lưu dữ liệu thô Accelerometer, Gyroscope và Temperature

    + `Ví dụ` : `ICM42688_ReadRawData(&raw_data)`

* `ICM42688_ConvertRawToSensorData(const ICM42688_RawData_t *raw_data, ICM42688_Data_t *sensor_data)` : Hàm chuyển dữ liệu thô thành giá trị gia tốc, vận tốc góc và nhiệt độ

    + `raw_data` : địa chỉ vùng dữ liệu thô Accelerometer, Gyroscope và Temperature

    + `sensor_data` : địa chỉ vùng nhớ dùng để lưu dữ liệu sau khi chuyển đổi

    + `Ví dụ` : `ICM42688_ConvertRawToSensorData(&raw_data, &sensor_data)`

* `ICM42688_ReadSensorData(ICM42688_Data_t *sensor_data)` : Hàm đọc dữ liệu thô và chuyển sang giá trị gia tốc, vận tốc góc và nhiệt độ

    + `sensor_data` : địa chỉ vùng nhớ dùng để lưu dữ liệu cảm biến

    + `Ví dụ` : `ICM42688_ReadSensorData(&sensor_data)`
